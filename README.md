# AppLimit

A Windows desktop application to limit application usage time.

## Features

- Add application rules
- Time limit per application
- Warning notification before limit expires
- Automatically close application
- Enable / Disable rules
- Usage timer
- SQLite local database

## Built With

- C#
- WPF
- .NET 10
- SQLite
- MVVM Toolkit
